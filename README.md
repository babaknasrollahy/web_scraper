# Stack_mag
This app is a complete articles **scraper** . 


# Technologies and Tools
This app has 4 Technologies and Tools:
1) Flask (Python)
2) Selenium (Python)
3) Docker and Docker-Compose (Container-Runtime)
4) MySQL (DataBase)

# Startup and Run
To run this app, just use **one** command :
```
docker-compose up
```
